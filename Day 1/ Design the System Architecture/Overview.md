# System Architecture Overview – Mind Map Concept

Before prototyping, it is important to design the system architecture in a simple mind map form. The goal is to understand **how the system works, why each component is used, and why the flow makes sense**. This ensures that every part of the wearable seizure detection system contributes effectively to the overall function.

## Purpose of This Architecture
- Ensures **clear flow from signal acquisition → processing → alert**
- Explains **why each component is necessary**
- Guides **system design decisions before building the prototype**
- Helps in **visualizing interaction between hardware, software, and user**
